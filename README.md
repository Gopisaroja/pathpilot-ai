# PathPilot AI 🚀

AI-powered Career Agent that evaluates resumes like a FAANG recruiter.

## Features
- AI Resume Scoring (0–100)
- ATS Optimization Engine
- FAANG-level feedback system
- Personalized Career Roadmap
- Study Planner + Mock Interviews
- Fallback system (offline safe mode)

## Tech Stack
- Next.js 14
- TypeScript
- Gemini API
- Vercel

## AI System
Uses prompt-engineered recruiter model to simulate:
- Google-style resume evaluation
- Skill gap detection
- Career readiness scoring
